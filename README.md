##Introduccion
En el desarrollo de sistemas modernos, es común combinar distintos lenguajes de programación para aprovechar las fortalezas de cada uno. Los lenguajes de alto nivel, como Python, facilitan la creación de interfaces y la manipulación de datos, mientras que lenguajes como C permiten un control más cercano al hardware. Por otro lado, el lenguaje ensamblador ARM64 ofrece un nivel de ejecución aún más bajo, permitiendo optimizar operaciones críticas directamente en el procesador.

El objetivo de este proyecto es integrar Python, C y ARM64 Assembly en un solo sistema funcional mediante el uso de librerías compartidas (.so) y la interfaz ctypes. A través de esta integración, se implementarán operaciones básicas como suma, resta, máximo y manejo de arreglos directamente en ensamblador, utilizando C como puente entre el bajo nivel y Python como interfaz principal.

Además de la implementación funcional, el proyecto busca analizar y comparar el rendimiento entre los tres niveles de abstracción, permitiendo comprender cómo las decisiones de implementación afectan la eficiencia del sistema. También se emplearán herramientas como Makefile para la automatización de compilación y GDB para la depuración del código.

En conjunto, este proyecto permite fortalecer el entendimiento de la interacción entre lenguajes de programación de alto y bajo nivel, así como la importancia de la optimización y la arquitectura del software en sistemas computacionales.
